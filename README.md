# 🗓️ Ajandam - Premium Digital Planner & Diary

Tam donanımlı, modern ve kişiselleştirilmiş bir Verimlilik (Productivity) uygulaması. Günlük görevlerinizi organize edebilir, haftalık hedeflerinizi kuşbakışı takip edebilir ve sadece size özel, arka plan şifrelemesiyle korunan dijital günlüğünüzü tutabilirsiniz.

## ✨ Özellikler

- **📅 Günlük Planlayıcı:** Görev ekleme, silme ve tamamlama (Kategori ve Öncelik bazlı).
- **📊 Verimlilik Analizi:** Günlük tamamlanan görevlere göre canlı güncellenen ilerleme çubuğu.
- **🗓️ Haftalık Görünüm (Kanban):** Seçilen tarihin bulunduğu haftayı otomatik hesaplayıp 7 günlük planı tek ekranda sunma.
- **🎯 Haftalık Odak:** O haftanın en önemli hedeflerini belirleme ve kaydetme alanı.
- **🔒 Gizli Günlük (Secret Diary):** Backend (Node.js) doğrulamalı kilit ekranı ile korunan, geçmişe dönük okunabilen kişisel defter.
- **🌙 Premium Karanlık Mod (Dark Mode):** Göz yormayan, derinlikli özel tasarım (Kullanıcı tercihi `localStorage` ile hatırlanır).

## 🛠️ Kullanılan Teknolojiler

- **Frontend:** HTML5, CSS3 (Modern UI/UX, Flexbox/Grid), Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **Veritabanı:** SQLite3
- **Mimari:** RESTful API

## 🚀 Kurulum ve Çalıştırma

Projeyi kendi bilgisayarınızda (Localhost) çalıştırmak için aşağıdaki adımları izleyin:

1. Projeyi bilgisayarınıza klonlayın.

2. Proje dizininde terminali açıp gerekli kütüphaneleri yükleyin:
    npm install

3. Sunucuyu başlatın :
    npm run dev 

4. Tarayıcınızdan http://localhost:5000 adresine giderek uygulamayı kullanmaya başlayabilirsiniz.
